# Logplex client (work in progress)

## Usage

```
cloud mycloud  # or export LOGPLEX_ENDPOINT
export LOGPLEX_AUTH_KEY=...

logplex-cli channel create
logplex-cli channel destroy
logplex-cli drain add
logplex-cli drain remove
```
